## Process

To start an item:
1. Create `exec-plans/active/YYYY-MM-DD-<slug>.md` with goal, approach, decision log, checklist
2. Remove the item from this file (or mark in-progress)
3. Reference the plan file in commit/PR messages
4. When done: move plan to `exec-plans/completed/` with a one-line summary
